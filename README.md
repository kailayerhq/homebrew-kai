# homebrew-kai

Homebrew tap for [Kai](https://kailayer.com) — semantic infrastructure for code change.

## Install

```bash
brew install kailayerhq/kai/kai
```

## Usage

```bash
kai init
kai capture
kai status

# MCP server for AI coding assistants
kai mcp serve
```
